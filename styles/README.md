# `styles/`

Feuilles de style additionnelles éventuelles (animations, styles d'impression,
overrides spécifiques).

> Le style global et le thème (palette verte, tokens shadcn/ui) vivent dans
> [`app/globals.css`](../app/globals.css). Ce dossier reste disponible pour des
> styles complémentaires si le besoin se présente.
